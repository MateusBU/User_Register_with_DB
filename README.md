# Commands
## *npm i create-next-app*  
*Installs the `create-next-app` package locally in your project (adds it to `node_modules`).*  
Useful if you plan to reuse it multiple times or script it.

## *npx create-next-app user_register*  
*Runs `create-next-app` without installing it permanently, and creates a new Next.js project in a folder named `user_register`.*  
Downloads and uses the latest version directly from npm.
